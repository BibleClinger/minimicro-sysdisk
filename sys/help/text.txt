Basic text-related commands:
   `text.clear` -- clear the text display
   `print s` -- print contents of `s`
   `input(prompt)` -- print prompt, then return user input
   `text.color = color.yellow` -- change text foreground color
   `text.inverse = true` -- set inverse mode
   `text.row = 10` -- put the text cursor on row 10
   `text.column = 50` -- put the text cursor on column 50
