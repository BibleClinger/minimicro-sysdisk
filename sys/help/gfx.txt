Basic graphics-related commands:
   `gfx.clear color.navy` -- clear graphics (color defaults to black)
   `gfx.color = color.purple` -- set the color for subsequent drawing
   `gfx.line x1,y1,x2,y2` -- draw a line from `x1`,`y1` to `x2`,`y2`
   `gfx.fillRect left,bottom,width,height` -- fill a rectangle
   `gfx.fillEllipse left,bottom,width,height` -- fill an ellipse
